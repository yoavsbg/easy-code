Chrome DevTools: Debugging popup, focus object and more..

Posted by Yoav Sabag on November 06, 2022

Most of the time they works, but when you have an issue with focus state it can be very frustrating to debug this state.
You can find the focus state in many components like popup when you are hovering for example.

So, how to debug it?!

There are many ways to debug focus, but I found this one the easiest for me :)

1. Open DevTools and place the breakpoint where you want to start.
2. Go to console tab and place the following code:

setTimeout(function(){debugger;}, 5000)

3. Go to your Popup or View and hovering on it.
(in case 5 seconds in not enough time, you can change it)
